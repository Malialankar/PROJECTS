#  Customer Purchase Behavior & Revenue Analysis Using SQL

## Project Overview --

This project focuses on analyzing customer purchase data to extract meaningful business insights using SQL. The analysis covers customer behavior, revenue trends, discount impact, and segmentation to support data-driven decision-making in a retail environment.

---

## Objectives --

* Analyze total revenue across different customer segments
* Understand the impact of discounts on purchasing behavior
* Identify top-performing products based on ratings and sales
* Segment customers based on purchase history
* Compare spending patterns across shipping types and subscription status

---

## Dataset Description --

The dataset contains customer transaction details, including:

* Customer demographics (age, gender, location)
* Product details (item purchased, category, size, color)
* Transaction data (purchase amount, payment method, shipping type)
* Customer behavior (previous purchases, frequency of purchases)
* Feedback (review rating)
* Subscription and discount information

---

## Key Business Questions Solved --

1. Total revenue generated by male vs female customers
2. Customers who used discounts but spent above average
3. Top 5 products with highest average review ratings
4. Comparison of purchase amount across shipping types
5. Spending behavior of subscribed vs non-subscribed customers
6. Products with highest discount usage rate
7. Customer segmentation (New, Returning, Loyal)
8. Top 3 products in each category
9. Relationship between repeat buyers and subscription
10. Revenue contribution by age group

---

## Tools & Technologies --

* SQL (MySQL)
* Aggregate Functions (SUM, AVG, COUNT)
* Window Functions (ROW_NUMBER)
* CASE Statements
* Common Table Expressions (CTEs)

---

## Key Insights --

* Identified high-value customer segments contributing maximum revenue
* Found that repeat customers are more likely to subscribe
* Highlighted top-performing products based on ratings
* Analyzed how discounts influence customer spending
* Discovered purchasing patterns across different demographics

---

## How to Use --

1. Import the dataset into MySQL
2. Run the SQL queries provided in the project file
3. Modify queries to explore additional insights

---

## Conclusion --

This project demonstrates how SQL can be used to perform end-to-end data analysis, from raw data exploration to generating actionable business insights.
